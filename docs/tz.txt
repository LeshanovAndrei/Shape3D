Shape3D
Бушуев Данил, Лешанов Андрей

Написать приложение для представления графических примитивов в трехмерном пространстве.
Использовать интерфейс и технологии OpenGl и сопутствующие библиотеки.
Язык - C++

Предусмотреть реализацию разных примитивов, например, куб, сфера, призма...
Фигуры должны уметь двигаться, менять цвет и размер. Фигуры можно объединить в агрегаты.
Камера центрирована на центр поля, может вращаться, расположена в верхней полусфере.
По F1 - справка

Необязательно, по возможности:
Возврат примитива в состояние по умолчанию
Запись состояния в файл
